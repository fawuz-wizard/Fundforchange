How to Use FundsforChange


Post Login/Sign Up Navigation

The first page displays the login page.Once users log in or sign up on FundsforChange, they will be directed to the the main home page which will display the main feed. The home page includes:

  • A bottom navigation bar for quick access to features
  • A clickable drop down button for additional options
  • A notification bell that displays platform notifications

Bottom Navigation Bar

  • Home:It is the main feed where users can post fundraising campaigns to and view current fundraising campaigns. To make posts here the user uses the create post section. It also contains the dropdown button at the top right where a user can find additional settings.

  • Chat: Allows users to message campaign organizers directly, ask questions, or get more information before donating via real-time chat.

  • Create Post: Users can start a fundraising campaign or share approved updates to the home feed or the feedback feed. It has a prompt box users can write updates or posts and a provision for attaching a photo/video file not exceeding 30mb.

  • Account: Displays personal credentials such as user picture, name, country, city, contact information, and a logout button.

Category Pages

This page categorizes fundraising campaigns so users can reach a particular demographic.Clicking on a campaign from the feed or searching by category takes users to a detailed Campaign Page. Features include:

  • Campaign Details: Title, description, images, videos, and fundraising goal.

  • Donation Options: Secure payment gateway for donations.

  • Contact the Campaign Organizer: Chat with the organizer directly.

  • Progress Bar: Shows the current fundraising status.

Dropdown Menu

  • Feedback: Showcases success stories from users who have benefited from the platform, with detailed accounts and media shared by campaign creators.

  • Terms and Conditions: Outlines the legal framework, Eligibility requirements, user responsibilities and fundraising campaign guidelines.

  • About: Contains the mission and vision statements of the organization, as well as the team that brought the platform to life.

  • How to use: Helps the user with information on how to navigate the platform.

  • Settings: Users can set platform preferences here.

Notification Bell

  • Provides users with chat,donations,commeents alerts etc
